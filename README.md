# Jamium

Music learning and composition platform with real-time chord analysis, visualizations, and AI tools.

## Dev

```bash
cd frontend
npm run dev        # Full stack (opens http://jamium)
npm run jamium     # Frontend only
```

## Stack

- **Frontend:** Next.js 16, React 19, Tailwind, TanStack Query, Tone.js
- **Backend:** Go REST API
- **Database:** Supabase
- **Deploy:** Vercel (frontend), Hostinger VPS (backend)
